Возможности расширения

🔐

Автошифрование внутри диалогов

Сообщения автоматически шифруются прямо внутри диалогов мессенджеров. Никто, кроме вас и получателя, не сможет прочитать переписку.

💾

Хранение на устройстве пользователя

Все ключи шифрования и данные хранятся исключительно на вашем устройстве. Никаких серверов — полная приватность.

📦

Возможность перенести данные

Безопасная синхронизация между устройствами (скоро поддержка android/iOS)

👤

Несколько профилей для работы

Создавайте отдельные профили с разными ключами шифрования. Разделяйте личную и рабочую переписку.

🔒

Шифрование текстов вне мессенджеров

Шифруйте любой текст прямо в браузере — для заметок, документов или отправки через любые каналы связи.

⚡

Мгновенная работа

Расширение работает в реальном времени, не замедляя мессенджер. Шифрование происходит незаметно для пользователя.

🛡️ Ваша переписка под защитой

АнтиМакс использует современные алгоритмы шифрования для обеспечения полной конфиденциальности ваших сообщений.

🚀 Установка расширения

Выберите удобный способ установки

Установить из Chrome Web Store
1

Скачайте архив

Разархивируйте после скачивания.

Скачать → Исходный код
2

Откройте страницу расширений

В браузере Chrome или Яндекс перейдите по адресу:

chrome://extensions
3

Загрузите расширение

Включите "Режим разработчика" в правом верхнем углу, нажмите "Загрузить распакованное расширение" и выберите разархивированную папку.

4

Делитесь с друзьями! 💜

Расскажите близким о безопасном общении. Вместе мы делаем интернет приватнее!
